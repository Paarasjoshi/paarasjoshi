I'm a backend engineer working on a B2B SaaS product. Most of my work is around REST APIs, databases, and keeping things running reliably in production. I care about writing code that's straightforward to maintain and fast enough that users don't notice it.

---

## Work

- Reduced API response time from ~1300ms to ~40ms by fixing slow queries and trimming unnecessary logic
- Improved database performance by ~30% through indexing and query rewrites
- Built and maintained backend services that serve 100K+ active users
- Designed and implemented notification pipelines using NSQ
- Handled production deployments — no data loss, no surprises
